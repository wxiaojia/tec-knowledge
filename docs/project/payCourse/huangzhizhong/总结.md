## 一、动机是什么
话题是什么：他想告诉我什么，他的动机是什么，
内在动机：对他产生了什么影响
外在动机：他想呼吁什么，想告诉我什么呢，我有什么可以借鉴的呢

## 二、目的是什么

- 想学习：强调重要性                    教我们学习什么
- 想获得启发：强调可迁移                 对我们有什么启发
- 想得到乐趣                           有什么娱乐的内容吗
- 想见你    
- 想遇到同好：背景相似、态度相似          是不是有什么跟我一样
